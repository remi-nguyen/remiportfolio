NOK-raise

Developped in Python, NOK-raise automates tasks on network devices like routers, switches and network termination equipments (NTE) from different constructors such as CISCO, HUAWEI and RAD.
Given an IP address and a Vlan ID, the script outputs status of the Spanning Tree Protocol and MPLS for specified VLAN then it connects to the neighboring devices and executes same tasks.
